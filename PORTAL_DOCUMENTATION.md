# Portal Documentation

# Overview

The Urban Air Quality Intelligence Platform provides a web-based interface for monitoring environmental conditions, visualizing air quality data, and supporting intelligent decision-making. The portal offers separate interfaces for administrators and citizens, enabling role-based access to environmental information.

---

# Portal Modules

## Administrator Portal

The administrator portal provides comprehensive environmental monitoring and analytics.

### Features

- Secure Login
- Live AQI Dashboard
- Weather Monitoring
- Pollutant Analysis
- Historical Environmental Data
- Interactive GIS Map
- AI Diagnostics
- Source Attribution Dashboard
- What-if Emission Simulator
- Decision Support Interface

---

## Citizen Portal

The citizen portal allows users to view environmental information in a simplified format.

### Features

- Secure Login
- Current AQI
- Weather Conditions
- Pollutant Information
- Health Advisory
- Environmental Awareness Dashboard

---

# Dashboard Overview

The dashboard presents environmental intelligence using interactive visualizations.

## AQI Monitoring

Displays:

- Current AQI
- AQI Status
- Air Quality Category
- Health Impact

---

## Pollutant Monitoring

Displays concentrations of:

- PM2.5
- PM10
- NO₂
- SO₂
- CO

---

## Weather Monitoring

Displays:

- Temperature
- Humidity
- Wind Speed
- UV Index
- Hourly Forecast

---

## GIS Map

The portal provides an interactive geographic map displaying monitoring locations and environmental information.

# Source Attribution

Displays estimated contributions from:

# What-if Simulation

Allows administrators to visualize the environmental impact of emission reduction strategies.

- Reduce traffic emissions
- Industrial emission control
- Biomass burning reduction
